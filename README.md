## Перед запуском проекта

```bash
npm install   # установка зависимостей проекта
```

## Для запуска проекта
```bash
  gulp
```

## Структура

```bash
build/               # Папка сборки
source/              # Исходники
  fonts/             # Шрифты
  images/            # Изображения
  js/                # Скрипты
  styles/            # Служебные стилевые файлы SCSS
```

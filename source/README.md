## Перед запуском проекта
### `npm install`

## Для запуска проекта
### `gulp`

## Структура

```bash
build/               # Папка сборки
source/              # Исходники
  fonts/             # Шрифты
  images/            # Изображения
  js/                # Скрипты
  styles/            # Служебные стилевые файлы SCSS
```
